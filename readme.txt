
1)Pour compiler : make

2)Lancer l'executable audioFX

3)Le nom des pédales éteintes sont en rouge, celles allumées sont en vert
Pour activer/désactiver une pédale, il suffit de cliquer sur son nom et d'appuyer sur un raccourci clavier (f->fuzz, g->flanger, w->wah, e->echo, c->chorus, v->vibrato, s->shelving, o->overdrive, t->tremolo, r->reverb)

4)Pour modifier les paramères des effets :
clic droit sur la nom de l'effet.
La couleur se change en blanc.
La modification de fait avec les flèches directionnelles.

Modification :
 - fuzz : gauche/droite -> gain, haut/bas -> mix (signal entrée/distordu)
 - overdrive : gauche/droite -> Intensité
 - echo : gauche/droite -> retard, haut/bas -> gain
 - flanger : gauche/droite -> retard, haut/bas -> fréquence du flanger
 - vibrato : gauche/droite -> retard, haut/bas -> fréquence
 - Shelving : t -> treble, b -> bass, a->allpass, gauche/droite -> fréquence, haut/droite -> gain
 - chorus : gauche/droite -> retard, haut/bas -> gain
 - Wah : gauche/droite -> fréquence de modualtion
 - Tremolo : gauche/droite -> fréquence de modulation, haut/droit -> mixage entrée/tremolo

5) Une documentation est disponible dans Documentation/html/index.html
